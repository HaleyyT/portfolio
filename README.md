# Haley Tran Portfolio

Recruiter-focused portfolio for Ly My Hoa (Haley) Tran, built as a static Vercel-ready site with a deep navy night-sky theme, animated fireflies, project highlights, resume links, and a local profile chatbot. 

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

Import `https://github.com/HaleyyT/portfolio.git` in Vercel. The project uses Vite, so Vercel can build it with:

- Build command: `npm run build`
- Output directory: `dist`

## Notes

The chatbot is intentionally client-side and grounded in curated resume/project facts. It does not require an API key and does not send recruiter questions to a server.
